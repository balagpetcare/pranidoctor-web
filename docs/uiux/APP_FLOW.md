### 1.2 Core Flow Principles

| Principle | Implementation |
|-----------|----------------|
| **3-Tap Rule** | Critical actions reachable in ≤3 taps |
| **Offline-First** | Core features work without network |
| **Voice-Ready** | All screens support voice commands (future) |
| **Low-Literacy** | Icons + voice > text wherever possible |
| **Emergency Priority** | Emergency button always visible |
| **Bangla-First** | Default language is Bengali |
